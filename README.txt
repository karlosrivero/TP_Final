Herramienta de visualizacion de acciones.

Desde el menu principal se pueden ejectuar cuatro comandos.

GET:        Comando para traer datos de un ticker de la api. Se pide el nombre del ticker, seguido de las fechas de intervalo en formato YYYY-MM-DD.
            Si el ticker y las fechas son validas, la informacion extraida de yfinance se guarda en ticker_data.db.

CHECK:      Comando para verificar si existen datos guardados para un ticker en particular.

DISPLAY:    Comando para visualizar los datos almacenados de un ticker en particular.

EXIT:       Comando para cerrar el programa.
